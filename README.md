## 🚀 Quick Start

### Option A — Auto Install (Recommended)
1. Extract this ZIP on your Windows PC
2. Double-click `INSTALL-AND-BUILD.bat`
3. Wait 3–5 minutes (downloads Node.js if needed, builds EXE)
4. Run `dist\ANON-IRAN Setup 5.0.0.exe` or `dist\ANON-IRAN-Portable-5.0.0.exe`

### Option B — Manual Build
Requirements: Node.js 18+ from https://nodejs.org
```
npm install
npm run build
```

### Option C — Dev Mode (no build)
```
npm install
npm start
```
Or double-click `RUN-DEV.bat`
